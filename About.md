# Smart-Fan
For this project I created an oscillating fan that is temperature controlled and has a light attached that is controlled by the light level in the room. The fan speed can also be adjusted by using a button. The goal of this prototype was to create a system that automatically regulates airflow to improve room comfort, because I often struggle with maintaining a comfortable overall temperature in my room.  Ultimately, I wanted to create a contraption that could detect a certain temperature and react by adjusting airflow accordingly, in which, reducing manual work. Since this devise would be placed on my desk, I added a LED Light to serve as a function desk lamp.
